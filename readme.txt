== Description ==

Woo Product Coming Soon will add a new functionality to display when the plugin will be back in stock. It will  add new fields to product inventory to display
custome date when the product will be in stock and custom note to be displayed on product single page when product is out of scope.

== Installation ==
1) Upload the plugin.
2) Activate the plugin.
